# Демо-пролог v2: «Нижний Флекс: Унгерн-ран»

## Что изменилось (по сравнению с v1)
- В школе остаются: Жека, Егор, Руфа, Даня Кумыс, Сава, Виталя, Тоха, Тос, Лёха.
- Вова, Саня, Буга, Шкарик вводятся только на пьянке.
- Егор живёт в частном доме: добавлен фон двора/крыльца.

## Как запустить
1) Установи Ren'Py.
2) Создай новый проект (любое имя).
3) Скопируй содержимое папки `game/` из архива в папку `game/` своего проекта (с заменой).
4) Сгенерируй/положи ассеты по путям из `docs/asset_list.csv`.
5) Run.
