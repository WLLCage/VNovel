# Style guide (арт) — без глича (v2)

## Общий стиль
- Полуреалистичная 2D-иллюстрация (visual novel), лёгкая фактура, без аниме-кавая.
- Современность: чистые, но не "глянцевые" цвета; мягкие тени; читаемые силуэты.
- Фоны: кинематографичный реализм, слегка "припылённый", без фотожёсткости.

## Разрешения
- BG: 1920×1080 (PNG или JPG q=85–92)
- Спрайты: PNG с прозрачностью, высота 1600–1800px (по пояс/по грудь)

## Консистентность спрайтов по фото
1) Сначала делаем эталон (Янас neutral).
2) Остальных гоним в том же стиле и кадре, используя эталон как reference.
3) Эмоции делаем от базового neutral (чтобы не "плавило" лицо).

## Промпт-шаблон (img2img по фото)
«Convert the provided photo into a visual novel character sprite, half-body, semi-realistic 2D illustration, soft shading, subtle texture, consistent lighting, no anime, no caricature, transparent background. Keep facial features recognizable. Match the style of the reference sprite.»

## Промпт-шаблон для BG
«Cinematic background, modern Russian private house interior / school interior, realistic but illustrated, soft natural light, subtle texture, no people, 1920x1080, visual novel background»
